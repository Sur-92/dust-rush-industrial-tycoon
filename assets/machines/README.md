# Approved cabinet-shop machine art

These original sprites are the approved visual baseline for the first Dust Rush customer.

| File | Machine | Intended footprint |
| --- | --- | --- |
| `table_saw.png` | Table saw | Compact, low, approximately 2×2 floor tiles |
| `planer.png` | Thickness planer | Compact and tall, approximately 2×2 floor tiles |
| `wide_belt_sander.png` | Wide-belt sander | Long, approximately 3×2 floor tiles |

## Shared specification

- 1024×1024 RGBA PNG
- real transparent background
- fixed 2:1 isometric view
- upper-left lighting
- cool slate-blue industrial palette with restrained amber safety accents
- original art created for Dust Rush; no third-party source imagery or commercial-game assets

Keep proportions and camera orientation intact when scaling. Scale and offset each texture in its presentation scene rather than destructively cropping the source asset. Selection feedback should remain a UI treatment—such as the existing floor ring, label color, or a restrained outline—and should not permanently recolor these source files.
